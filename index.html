Projects

Personal projects I have been working on...
Christmas MIDI Piano Light Show
December 2017 • Serial Communication / User Interactive / C++

In essence it's a Christmas light show controlled by a piano or computer. Specifically, I designed and constructed a fully functional and programmable light control system consisting of up to 32 independently controlled channels. The unit could be pre-sequenced through an open source platform called Vixen or controlled instantaneously via a MIDI piano allowing for an user-interactive experience (its current state and most successful state).

SMS SCADA
January 2017 • Cellular Communication / Product Design / Arduino

Remotely accessible device that could gather information about location and sensor data, but also allow the user to control real world outputs connected to the device such as a relay to shut off or disable a car (remote kill-switch). The device could be expanded to industrial applications in remote locations lacking infrastructure.

1988 For Bronco Resto-Mod
October 2016 • Mechanical Repair / Diagram Interpretation / Electrical

This is a purely for fun project that started with the purchase of a 1988 Ford Bronco that had a partially complete engine that and would not run. Since then the truck has undergone an engine swap and a complete suspension rebuild/upgrade greatly increasing wheel travel (Long-Travel Kit). The bronco also has the SMS SCADA Project installed as an anti-theft device where the truck can be remotely tracked and shutdown with a cell phone. In the future it will also implement remote start with text.
